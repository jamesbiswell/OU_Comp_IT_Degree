## OU Degree - Computing & IT

- This repository contains the learning materials for the level 2 and 3 modules.

- It also contains a Jupyter Notebook using LangChain, OpenAI and Pinecone to create (and test querying) the knowledge base that is stored in Pinecone.

- The Streamlit RAG Q&A App provides a simple GUI to interact with the knowledge base.

### Module Summary:

- M255: Object-Oriented Programming with Java
- M257: Putting Java to Work
- M256: Software Development with Java
- M263: Building Blocks of Software
- M359: Relational Databases: Theory and Practice
- M362: Developing Concurrent Distributed Systems
- M363: Software Engineering with Objects
- TM354: Software Engineering
